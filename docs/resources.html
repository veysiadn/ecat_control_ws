Useful Resources
In this page you can find relevant resources on the EtherCAT protocol, the IgH EtherCAT library, EtherCAT master, real-time systems and performance optimization methods.

EtherCAT Protocol Related Resources

  • EtherCAT Webpage is maintained by EtherCAT Technology Group (ETG), and provides an in-depth introduction into EtherCAT (Ethernet for Control Automation Technology).
  • Industrial Communication Technology Handbook , Chapter 18 covers EtherCAT protocol in detail, additionally it covers available open-source libraries for EtherCAT master implementation.
  • Real-time Servo Control using EtherCAT Master on Real-time Embedded Linux Extensions presents a real-time servo control application based on an open-source EtherCAT Master under two different real-time embedded Linux approaches, the fully preemptible kernel and Xenomai.
  • IgH EtherCAT Library Documentation explains the design of the library and details the implementation.
  • Maxon EPOS Firmware Specifications provides information about CiA 402 motion control standard, object dictionary elements of EPOS drivers, allowable PDO mappings and operation modes in detail.

Real-time Related Resources

It is CRUCIAL that you comprehend the real-time concept, for that I strongly recommend reading the first two resources mentioned below.
  • Challenges Using Linux as a Real-Time Operating System highlights aspects of the Linux Kernel that require special attention for real-time applications and covers more topics than addressed by the Build an RT-application HOWTO of the Real-Time Linux Wiki.
  • Enea® Linux Real-Time Guide document tries to guide anyone attempting to implement a real-time application using Linux.
  • Real-time Linux covers the technical basics and important aspects of real time systems.
  • The Real-Time Linux Kernel: A Survey on PREEMPT_RT provides a survey on the state-of-the-art approaches for building real-time Linux-based systems, with a focus on PREEMPT_RT, its evolution, and the challenges that should be addressed in order to move PREEMPT_RT one step ahead.
  • Presentation on Real-Time Systems covers starting from definitions of real-time systems to scheduling mechanisms in real-time systems.
  • Real-Time Performance Analysis in Linux-Based Robotic Systems presents best practices while using real-time Linux (RT_PREEMPT) patch and provides timing measurements of each practice.
  • Real-time Background similiarly presents the best real-time programming practices.
  • Core Partitioning Techniques on Multicore Linux Systems covers CPU core isolation to dedicate cores to specific application.
